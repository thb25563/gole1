**题目描述**

给定一个整数数组 nums，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。你可以按 任意顺序 返回答案.
注意：如能实现O(1)空间复杂度最佳.

**示例**

示例1：

输入：nums = [1,2,1,3,2,5]

输出：[3,5]

解释：[5, 3] 也是有效的答案。

示例2：

输入：nums = [-1,0]

输出：[-1,0]

示例3:
输入：nums = [0,1]

输出：[1,0]

**前提条件**

- 2 <= nums.length <= 3 * 10^4
- -2^31 <= nums[i] <= 2^31 - 1
- 除两个只出现一次的整数外，nums 中的其他数字都出现两次
