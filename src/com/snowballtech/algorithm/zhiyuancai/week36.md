**题目描述**

(148) 给你链表的头结点 `head` ，请将其按 **升序** 排列并返回 **排序后的链表** 。

要求能够达到O(nlogn)的时间复杂度和常数级空间复杂度



**示例**

示例1：

![sort_list_1](.\img\sort_list_1.jpg)

```
输入：head = [4,2,1,3]
输出：[1,2,3,4]
```

示例2：

![sort_list_2](.\img\sort_list_2.jpg)

```
输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]
```

示例3：

```
输入：head = []
输出：[]
```



**前提条件**
$$
链表中节点的数目在范围 [0, 5 * 10^4] 内
$$

$$
-10^5 <= 节点值 <= 10^5
$$
